# Documentation Rules

Standards for writing docs in this project.

## Location

- All research and reference docs live in `docs/`.

## Bilingual documents

- When a document is meant for a bilingual audience, produce two files:
  `<name>.pl.md` (Polish) and `<name>.en.md` (English).
- Keep both versions structurally identical (same sections, same order).
- Cross-link the versions at the top of each file.

## Research documents

- Separate primary sources (product/app) from peer-reviewed research in the Sources section.
- Flag marketing/affiliate content and unverified claims explicitly.
- Cite studies as: Author (Year). *Title.* Journal, volume(issue), pages.
